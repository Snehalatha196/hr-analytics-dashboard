## HR Analytics Dashboard Project

Project Overview
This project analyzes HR employee data to identify key workforce insights such as employee distribution, salary trends, and attrition rate.

## Tools & Technologies Used
- Microsoft Excel (Data Cleaning)
- SQL (Data Analysis & Queries)
- Power BI (Interactive Dashboard & Visualization)

## Key Insights
- Total Employees: 50
- Average Salary: 59.93K
- Attrition Rate: 26%
- Department-wise Employee Distribution
- Gender-wise Analysis
